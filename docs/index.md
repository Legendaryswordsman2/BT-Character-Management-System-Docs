# BlazerTech Character Management System Docs

An all in one solution for handling 2d pixel art characters in the Unity Game Engine.

## Character Types
    Two types of characters are available:

    #1 Unified Character - A single spritesheet which contains the finished character.
    #2 Layered Character - A list of spritesheets with each containing a layer of the character (Ex: Body, Outfit, Hairstyle, Accessory)


## Commands

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.
